Sobre mim

Ola, sou estudante de Sistemas para Internet na Fatec - Lins.
Sou apaixondo por gatos e musica, no tempo livre gosto de jogar jogos online, viajar, escutar musica e fotografar paisagens

Sou desenvolvedor front-end com experiência em HTML, CSS e Angular, mas estou aberto para aprender e explorar novas tecnologias, mas tambem gosto das áreas de UX design e banco de dados.

Projetos

Projeto realizado no primeiro semestre, feito apenas com HTML e CSS, feito para aprender os conceitos basicos da programação. Durante o desenvolvimento, foram explorados fundamentos de design visual, semântica HTML e estruturação de páginas web.

mirrorFashion

Projeto desenvolvido no segundo semestre, consistindo em um site responsivo criado com HTML, CSS e o framework Materialize. O objetivo foi aplicar conceitos de design responsivo e organização visual, resultando em uma interface adaptável a diferentes dispositivos.

Yolo

Projeto realizado no terceiro semestre, consistindo em um CRUD de demandas de manutenção de computadores. Desenvolvido com HTML, CSS e JavaScript, permite cadastrar clientes, serviços e ordens de serviço, com cálculo automático de custos. O sistema utiliza Local Storage para armazenar os dados e aplica conceitos de programação orientada a objetos.

RedTech

Projeto desenvolvido no quarto semestre, criando um banco de dados relacional para uma gravadora musical. Inclui tabelas de artistas, álbuns, músicas, clientes e vendas, com chaves primárias, estrangeiras e índices. Foram realizadas operações de inserção, atualização, exclusão e consultas, permitindo organizar e consultar informações de forma estruturada e eficiente.

Gravadora de Musica

Projeto desenvolvido no quinto semestre, voltado ao planejamento de custos de viagem. Permite registrar despesas com transporte, hospedagem, alimentação e lazer, gerando estimativas totais. Desenvolvido com Ionic e Capacitor, o aplicativo oferece uma interface intuitiva e responsiva, unindo organização financeira e experiência do usuário.

Planejador de Viagem

Projeto desenvolvido no último semestre, voltado ao registro e gerenciamento de produtos em uma loja. Permite cadastrar, excluir e calcular preços finais com base em operações de aumento e desconto, exibindo o total da compra. Desenvolvido com Java e Swing, o sistema apresenta uma interface intuitiva e interativa, integrando conceitos de programação orientada a objetos e manipulação de dados em tabela.
